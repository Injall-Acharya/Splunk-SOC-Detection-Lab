# Datasets

This folder contains sample log data used for the Splunk SOC Detection Lab.

## Apache Access Logs

The Apache access log dataset contains web server activity including client IP addresses, HTTP requests, requested URLs, HTTP methods, and response status codes.

The dataset is used in Splunk Enterprise for:
- Log ingestion
- Security monitoring
- Threat detection
- Dashboard creation
- Alert generation

## Source

Public sample Apache web server logs used for educational security analysis.
